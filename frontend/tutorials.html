    Getting Started

    To begin, sign up via the Registration page. After logging in, explore the Dashboard, which gives you access to key features like searching for models, uploading your contributions, and discovering community-shared models.

    Uploading Your Own Models

    Follow these steps to upload your models:

    1. First, prepare your metadata in STAC-compliant JSON format using this template.
    2. Go to the Upload page, fill out the form, and upload your file.
    3. If you encounter issues, a specific message will pop up. Otherwhise just contact us.
    4. Successfully uploaded models can be found in your account.

    Integrating with Python and R

    Follow these steps to integrate models into your workflows:

    1. Install the required STAC client: pip install pystac for Python or install.packages('rstac') for R.
    2. Download model metadata from the Catalog page.
    3. Integrate it using tools like pystac.Item() in Python or stac_read() in R.
    4. Check out our example notebooks for practical usage.

    The HuggingEarth-Catalog is a platform that provides geoscientific models to browse, retrieve, and integrate data into workflows.

    Click on "Login" on the Login page and follow the instructions. A valid email address is required for registration.

    The platform offers various models for climate data analysis, environmental monitoring, and geodata integration. You can find a list of models on the Catalog page.

    Install the `pystac` library and use it to integrate data directly from the platform into your workflows. Example codes can be found in our PySTAC tutorial.
